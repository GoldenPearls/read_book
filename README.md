### 📖 책 읽자!!!!

- 독서를 읽고 기록한 블로그에 대한 링크를 정리하는 용 
- 조금씩이라도 꾸준히
- 강의도 좋지만 책으로 얻을 수 있는 지식 또한 많다.
- 1주일에 1단원~2단원 이상 읽는 것이 목표
- 무리하지 않는 선에서 읽기
